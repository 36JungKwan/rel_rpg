# rel_rpg

Mô tả
-----

`rel_rpg` là một dự án Python nhỏ cho một ứng dụng/mini-game liên quan đến cơ sở dữ liệu quan hệ (ví dụ: một bản demo RPG đơn giản). Tệp chính là `rel_rpg.py`.

Yêu cầu
-------

- Python 3.8 hoặc mới hơn
- (Tùy chọn) Các thư viện bên ngoài nếu mã yêu cầu — kiểm tra phần đầu của `rel_rpg.py` để biết chi tiết.

Cài đặt
--------

1. Sao chép hoặc clone repository này.
2. (Tùy chọn) Tạo virtual environment và cài đặt phụ thuộc:

```bash
python -m venv .venv
source .venv/bin/activate  # Linux / macOS
.venv\Scripts\Activate     # Windows (PowerShell)
pip install -r requirements.txt  # nếu có
```

Chạy chương trình
-----------------

Ví dụ lệnh đơn giản để chạy tệp chính:

```bash
python rel_rpg.py
```

Nội dung và ví dụ
------------------

- Mở `rel_rpg.py` để xem cách chương trình hoạt động và các tuỳ chọn cấu hình.
- Thêm ví dụ sử dụng hoặc tập dữ liệu ở đây nếu cần.
